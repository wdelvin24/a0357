üëî Updating Officer Names

Edit the file: index.html

Search for the officers section and replace [Name] with actual names:

<div class="officer"> <p class="title">Worshipful Master</p> <p class="name">[Name]</p> ‚Üê Change this </div>

Do this for each officer, then re-upload to Netlify.

üîê Changing Passwords

Edit the file: index.html

Search for this section near the bottom of the file:

const BROTHER_PASSWORD = "AO357"; // For all brothers const ADMIN_PASSWORD = "AO357Admin"; // For officers/admin

Change the text inside the quotes to your new passwords, then re-upload to Netlify.

üì∞ Updating the Newsletter

Edit the file: trestle-board.html

This is the most complex update. You'll need to edit HTML to change:

  • Worshipful Master's message
  • Upcoming events
  • Announcements
  • Birthdays
  • Memorials
  • Volume/Issue numbers

Open the file in a text editor (Notepad, VS Code, etc.) and look for the text you want to change. The content is mostly plain text inside HTML tags.

üí° Tip

Use "Find and Replace" (Ctrl+H or Cmd+H) to quickly update recurring text like dates or names.

üíæ Keeping the Backup ZIP Current

The "Download Site Files" link in the admin section gives officers a copy of the website files. After making updates, you should update this backup:

  1. Select all the website files (not including the old ZIP)
  2. Compress them into a new ZIP named ao357-site-backup.zip
  3. Include the new ZIP when you upload to Netlify

This ensures officers always download the latest version.
