Common Tasks

Adding Grand Lodge Communications

When you receive info from Grand Lodge (dress codes, deadlines, etc.):

  1. Create an event on the relevant date
  2. Title: Clear description (e.g., "‚ö†Ô∏è Dues Deadline - $165 Rate")
  3. Description: Paste or type all the details
  4. For dress codes: Create an event the day before the festive day

Adding Degree Work

  1. Title: "EA Degree" / "FC Degree" / "MM Degree"
  2. Include candidate info in description (if appropriate)
  3. Note any special requirements (attire, rehearsal times)

Adding Fundraisers

  1. Title: Clear name (e.g., "Annual Fish Fry")
  2. Include: Time, location, ticket prices, what's needed
  3. Add setup time as a separate event if needed

Meeting Cancelled?

  1. Click on the event
  2. Click Edit (pencil icon)
  3. Add "CANCELLED - " to the beginning of the title
  4. Add reason in description
  5. Or delete the single occurrence

To set color: When creating/editing an event, click the colored circle next to the calendar name.

Troubleshooting

Event not showing on website?

  • Make sure you saved the event
  • Refresh the website page
  • Check that you added it to the correct calendar (Alpha Omega Lodge No. 357 Events)

Can't sign in?

  • Make sure you're using the lodge Google account, not your personal one
  • Check with another officer for the correct password
  • Try "Forgot password" if needed

Accidentally deleted something?

  • Check the Trash in Google Calendar (Settings ‚Üí Trash)
  • Deleted events stay in trash for 30 days

Handing Off to the Next Person

When passing this responsibility to another officer:

  1. Share login credentials securely (in person, not via text/email)
  2. Walk through this guide together
  3. Have them add a test event while you watch
  4. Show them how to edit and delete
  5. Explain the color coding system
